> Операции, с которыми работает калькулятор - 
   1. Стандартные арифметически операции:
      +  -  *  /  ()  |x|
   2. Нестандартные арифметически операции:
      %(остаток от деления)  ^(возведение в степень)  !(факториал числа)  &(исключающее или)
   3. Триганометрические функции:
      sin()  cos()  tg()  ctg()
   4. Логарифмические функции:
      ln()  lg()
Работа с переменными и константами - 
   let name = 2     -  Создать переменную с именем 'name' и значением '2'
   name = 3         -  Присвоить переменной с именем 'name' значение '3'
   const name = 4   -  Создать константу с именем 'name' и значением '4'
Общие правила ввода выражений - 
   1. Чтобы вычислить некоторое выражение, введите его и нажмите Enter
   2. Чтобы вычислить несколько выражений подряд, введите их в одну строчку, разделяя символом ';'
   3. Введя переменную или константу, вы можете взимодействовать с ней, как с обычным числом, обращаясь по ее имени
